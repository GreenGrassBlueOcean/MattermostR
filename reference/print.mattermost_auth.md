# Print a mattermost_auth object

Displays the server URL and a masked version of the bearer token to
avoid accidentally exposing credentials in console output or logs.

## Usage

``` r
# S3 method for class 'mattermost_auth'
print(x, ...)
```

## Arguments

- x:

  A `mattermost_auth` object.

- ...:

  Additional arguments (ignored).

## Value

`x`, invisibly.
